# 🎰 Maszyna Losująca Projekty Programiaste

Kiczowaty i humorystyczny generator absurdalnych pomysłów na projekty programistyczne. Zastanawiałeś się kiedyś, co zaimplementować do swojego portfolio, na hackathonie, albo "do szuflady"? Ta maszyna wyręczy Cię w myśleniu, losując projekty, od których "senior usiądzie".
Projekt inspirowany twórczością **Klocucha** z YouTube oraz kultową **Maszyną do losowania tematów TV Gry**.

## ✨ Funkcje

- **Losowanie 3-częściowe:** Maszyna działa jak jednoręki bandyta i losuje:
  - **Typ projektu** (np. aplikacja webowa, sztuczna mądrość, terminalowy kombajn)
  - **Dla kogo** (np. dla seniora bez myszy, dla lodówki na websocketach)
  - **Po co** (np. który zamienia bugi w tapetę, który robi deploy tylko jak ładnie poprosisz)
- **Generowanie obrazka (Canvas API):** Wylosowałeś coś niesamowitego? Możesz wygenerować i pobrać gotowy obrazek ze swoim pomysłem - w pięknej oprawie graficznej przypominającej kiczowate automaty.
- **Śledzenie "Czystego Sumienia":** Aplikacja zapisuje w przeglądarce (LocalStorage) ilość wykonanych losowań i ocenia Twój stan - od "czystego sumienia" do "agencji kreatywnej po awarii".
- **Dynamiczne dopasowanie tekstu:** Tekst automatycznie dopasowuje się do okienek maszyny, więc nic nie ucieka z interfejsu.

## 🚀 Jak uruchomić?

To prosty projekt typu Vanilla Web. Nie wymaga żadnego etapu budowania ani instalacji paczek (zero npm, bundlerów i frameworków).

1. Pobierz pliki projektu na swój komputer.
2. Otwórz plik `index.html` bezpośrednio w swojej ulubionej przeglądarce.
3. Kliknij przycisk **"losuj"** i poczekaj na wymarzony projekt.

## 🛠 Technologie

- **HTML5** - semantyczna struktura aplikacji
- **CSS3** - stylowanie aplikacji, siatki, responsywność i animacje w CSS
- **Vanilla JavaScript (ES6+)** - logika losowania, manipulacja DOM, generowanie grafiki za pomocą Canvas API i zapisywanie statystyk w LocalStorage.

## ⚠️ Ostrzeżenie z instrukcji

> Maszyna nie bierze odpowiedzialności za frameworki, deploye, sprinty ani za to, że pomysł brzmi jak dobry pomysł.
